# Antonia
You can view this website at antonia.rf.gd

Antonia is a static website for an online women's clothing store. 
This website was built with the following frameworks and libraries:

- Bootstrap
- SweetAlert2
- Slick Js 
- Animate.css
- Animsition
- Select2
